# Restoran Sipariş ve Yönetim Sistemi

Masa takibi, sipariş yönetimi ve dinamik ürün kartları sunan, restoran işletmeleri için geliştirilmiş bir masaüstü otomasyon uygulamasıdır.

## 🚀 Kullanılan Teknolojiler
* **Platform:** Windows Masaüstü (Desktop)
* **Dil & Çerçeve:** C# / WinForms
* **Mimari:** Runtime UI

## 📌 Temel Özellikler
* Özelleştirilebilir ürün ve malzeme seçimi
* Anlık sipariş tutarı hesaplama
* Dinamik arayüz ile kolay masa ve sipariş takibi
